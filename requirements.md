Нужно написать консольное Python-приложение для управления книгами и читателями. БД-SQLite база храниться в одном файле Libraary.db

authors (авторы): id (PK), nаmе (имя).
books (книги): id (PK), title
(название), author_id (
FK к таблице авторов).
readers (читатели): id (PK), nаmе (имя), borrowed_book_id (FK k books, мож быть NULL, если книга не взята). Связь: один читатель может держать только одну книгу.

 Функционал программы (Интерфейс в консоли)

При запуске файла main.ру программа должна предлагать выбор действий (через input() и циклы):

1. Добавить автора (ввод имени).

2. Добавить книгу (ввод названия и выбор автора из списка существующих).

3. Добавить читателя.

4. Выдать книгу читателю (связать читателя с книгой, обновив borrowed_book_id).

5. Показать список всех книг и имён их авторов (тут ей понадобится JOIN ).

6. Показать список должников (вывести имена читателей и названия книг, которые они взяли).

4. Требования к реализации (Для контроля с вашей стороны)
